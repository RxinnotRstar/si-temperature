# si-temperature

基于 C++23 的强类型温度计算库。

新手上路，vibe coding 产物，大佬轻喷。

## 特点

### 基本

可以区分温度点和温度差。

支持用户通过字面量后缀直接指定单位，例如`_C`表示摄氏度，`_F`华氏度，`_C_diff`表示摄氏度为单位的温差等。

允许直接赋值浮点数和整数，这时候会视为摄氏度调用构造函数。

值得注意的是不允许`-5_C`这样的写法，没有重载负号。但是直接赋值`-5`是可以的，它被视为摄氏度。

这是刻意为之。如果你确定你的环境当前只有摄氏度，你可以不承担额外的编写负担，当作普通的`double`或是`int`用都可以。如果你后续涉及到其他模块使用华氏度甚至开尔文的问题，也可以无损兼容，而不担心单位不同带来的计算错误。

### 运算

支持：

- 温度点加减温度差
  - 在这种情况下，支持`+=`和`-=`。
  - 同时，允许右侧是整数或浮点数的形式的温度差。
- 温度点相减得到温度差
- 温度差乘以常数
- 温度差除以常数
- 允许对温度差取负
